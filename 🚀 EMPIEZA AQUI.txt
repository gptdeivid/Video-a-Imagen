───────────────────────────────────────────────────────────────────

   1. Instalar dependencias:
      pip install -r requirements.txt

   2. Iniciar servidor web:
      python app.py

   3. Abrir navegador en:
      http://localhost:5000


🔧 ARCHIVOS DEL SISTEMA (No modificar):
───────────────────────────────────────────────────────────────────

   app.py                 → Servidor web Flask
   extract_frames.py      → Script original (línea de comandos)
   requirements.txt       → Lista de dependencias
   templates/             → Carpeta con archivos HTML


📦 ¿QUÉ HACE ESTE PROGRAMA?
───────────────────────────────────────────────────────────────────

   Convierte videos en imágenes PNG individuales (frames).
   
   Ejemplo:
   - Tu video: "mi_video.mp4"
   - Resultado: Carpeta "mi_video_frames/" con:
     • frame_0000.png
     • frame_0001.png
     • frame_0002.png
     • ... etc.


✅ REQUISITOS:
───────────────────────────────────────────────────────────────────

   • Python 3.7 o superior
   • Conexión a internet (solo para la primera instalación)
   • Espacio en disco para los frames
